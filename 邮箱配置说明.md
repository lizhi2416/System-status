# 邮箱配置说明

## 新浪邮箱配置步骤

1. **登录新浪邮箱**
   - 访问：https://mail.sina.com.cn
   - 使用你的账号登录

2. **开启SMTP服务**
   - 点击右上角"设置"
   - 选择"账户"或"账户设置"
   - 找到"POP3/SMTP服务"选项
   - 开启SMTP服务
   - **重要**：如果提示需要授权码，请记录授权码（不是登录密码）

3. **更新配置文件**
   - 打开 `config.py`
   - 如果生成了授权码，将授权码填入 `password` 字段
   - 如果使用登录密码，确保SMTP服务已开启

4. **测试配置**
   ```bash
   python test_email.py
   ```

## QQ邮箱配置步骤（推荐，因为收件邮箱是QQ邮箱）

1. **登录QQ邮箱**
   - 访问：https://mail.qq.com
   - 使用你的QQ账号登录

2. **开启SMTP服务并获取授权码**
   - 点击"设置" -> "账户"
   - 找到"POP3/IMAP/SMTP/Exchange/CardDAV/CalDAV服务"
   - 开启"POP3/SMTP服务"或"IMAP/SMTP服务"
   - 点击"生成授权码"
   - **重要**：复制生成的16位授权码（不是QQ密码）

3. **更新配置文件**
   - 打开 `config.py`
   - 将授权码填入 `password` 字段
   - 确保 `from_email` 是你的QQ邮箱

4. **测试配置**
   ```bash
   python test_email.py
   ```

## 常见问题

### 认证失败怎么办？

1. **确认SMTP服务已开启**
   - 新浪邮箱：设置 -> 账户 -> POP3/SMTP服务 -> 开启
   - QQ邮箱：设置 -> 账户 -> POP3/SMTP服务 -> 开启

2. **使用授权码而不是登录密码**
   - 如果邮箱要求授权码，必须使用授权码
   - 授权码通常在邮箱设置中生成

3. **检查密码是否正确**
   - 确认密码中没有多余空格
   - 确认大小写正确

4. **尝试不同的端口**
   - 新浪邮箱：465（SSL）或 25（非加密）
   - QQ邮箱：587（TLS）或 465（SSL）
